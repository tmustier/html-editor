# html-editor Agent Skill

Explicitly launch the local collaborative editor for an HTML or SVG file when the user asks for interactive editing or commenting.

## Installation

Install the package so Pi loads both the explicit-invocation skill and its per-session comment bridge:

```bash
pi install git:github.com/tmustier/html-editor
```

See the [package README](../../README.md) for runtime dependencies, usage and development instructions.
